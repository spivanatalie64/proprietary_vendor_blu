# Proprietary Vendor Blobs for BLU Bold N2

Device: BLU Bold N2
Codename: bold_n2
Platform: MediaTek MT6833 (Dimensity 810)

## Source

Extracted from stock firmware: BOLD_N0050UU_V11.0.04.04_GENERIC_20220914_2257

## Contents

- Hardware Abstraction Layers (HALs)
- Firmware files
- MediaTek proprietary libraries
- Vendor binaries and configurations

## Usage

This repository is meant to be used with the device tree:
https://github.com/spivanatalie64/android_device_blu_bold_n2

Place in: `vendor/blu/bold_n2/` in your LineageOS source tree.
